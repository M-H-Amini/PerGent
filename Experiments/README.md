This folder contains the data and scripts for each research question.

## RQ1

| File                | Description                                                                                             |
| ------------------- | ------------------------------------------------------------------------------------------------------- |
| log_percentages.csv | Percentage of persona items generated by PerGent and the baselines that were edited, removed, or added. |

## RQ2

| File             | Description                                                                                                                           |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| measurements.csv | Raw measurements comparing LLM-generated personas with pre-LLM personas.                                                              |
| avg.csv          | Average scores for each method and LLM across all personas and five repetitions.                                                      |
| stat.csv         | Wilcoxon signed-rank test and Vargha-Delaney A12 effect size results comparing PerGent against each baseline for each LLM.            |
| average.py       | Computes the average scores for each method and LLM.                                                                                  |
| stat.py          | Runs the Wilcoxon signed-rank test and computes the Vargha-Delaney A12 effect size for comparisons between PerGent and each baseline. |

## RQ3

| File       | Description                                                                                                   |
| ---------- | ------------------------------------------------------------------------------------------------------------- |
| tokens.csv | Counts of input/output tokens and API calls for each method and LLM across all personas and five repetitions. |
| avg.csv    | Average number of tokens and API calls for each method and LLM across all personas and five repetitions.      |
| average.py | Computes the average number of tokens and API calls for each method and LLM.                                  |

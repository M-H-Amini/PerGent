# Sample Personas

This folder contains two example personas generated by PerGent for illustration purposes only.

They were produced using 5 publicly available job postings for each persona; no proprietary materials or resources from our industry partner were used in their creation.

Included personas:

- `demand_planner.md`
- `supply_planner.md`

These examples are provided solely for demonstration and should not be used as real personnel profiles.
